# TRIT-RV16K-SIM
TRIT RV16K Functional Simulator

# How to use
## Build
```
make
```
## Test Instruction RV16K
```
make test
```

## Use
```
Usage: ./main NCYCLES [-a] [-q] [-m] [-t ROM] [-d RAM] [FILENAME]
Options:
  -q       : No log print
  -m       : Dump memory
  -t ROM   : Initial ROM data
  -d RAM   : Initial RAM data
  -a       : TRIT-RV16K
  FILENAME : ELF Binary
```
